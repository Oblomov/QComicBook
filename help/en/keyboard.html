Keyboard reference

Ctrl+DOpen directory
Alt+IShow comicbook info
ArrowsScroll page contents
Shift + ArrowsScroll page contents fast
PageDownGo to next page
PageUpGo to previous page

Alt+WFit page width
Alt+HFit page height

F11Toggle fullscreen mode
ESCExit fullscreen mode
